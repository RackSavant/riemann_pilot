PROMPT 2: CONNECT TO API
==========================================

Connect the SistaChat interface to my backend API.

First, replace the API URL with your deployed Railway/Render URL:
const API_BASE = 'https://riemann-pilot.railway.app';  // UPDATE THIS

Create an API client in src/lib/sistaChat.ts with these TypeScript functions:

```typescript
const API_BASE = 'https://riemann-pilot.railway.app'; // Your API URL

export interface Character {
  character_id: string;
  character_name: string;
  appearance: string;
  dial_values: {
    theory_of_mind: number;
    harmfulness: number;
    irony: number;
    self_other: number;
  };
}

export async function getCharacters(): Promise<Character[]> {
  const response = await fetch(`${API_BASE}/api/characters`);
  const data = await response.json();
  return data.characters;
}

export async function generateRound(
  prompt: string,
  model: string = 'gpt-4',
  generateVideos: boolean = false
) {
  const response = await fetch(`${API_BASE}/api/conversation/round`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ prompt, generate_videos: generateVideos, model })
  });
  return response.json();
}

export async function updateDial(
  characterId: string,
  dimension: string,
  value: number
) {
  const response = await fetch(`${API_BASE}/api/dial`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ character_id: characterId, dimension, value })
  });
  return response.json();
}
```

When user clicks "Start Conversation":
1. Call generateRound() with the prompt and selected model
2. Show loading spinner for each character panel
3. Display responses as they arrive
4. Show validation scores below each response as progress bars
5. If VEO enabled, show video players
6. Handle errors with toast notifications

Load characters on page mount using getCharacters().
